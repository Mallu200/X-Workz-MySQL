# MySQL Installation & Setup

To practice MySQL, you need two main components:

1. **MySQL Community Server:** This is the "Engine" that sits in the background and processes the data.
2. **MySQL Workbench:** This is the "Visual Tool" where you actually type your code and see the tables.

### Setup Steps:
1. Download the **MySQL Installer** from the official website.
2. Choose the "Developer Default" setup.
3. **Important:** Set a "Root Password" you will remember. (Usually `root` or `admin123`).
4. Open **MySQL Workbench** and connect to `Local Instance 3306`.
